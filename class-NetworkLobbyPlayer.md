Network Lobby Player
==============

The NetworkLobbyPlayer is used by the NetworkLobbyManager to store per-player state while in the lobby.

Properties
----------

|**_Property:_** ||**_Function:_** |
|:---|:---|:---|
|__ShowLobbyGUI__ ||True to show the legacy user interface for players in the lobby. |
|__slot__ ||The slot that this player was allocated to. Only one player can be in each lobby slot. |
|__readyToBegin__ ||Flag to control if the player is ready for the game to start. |
