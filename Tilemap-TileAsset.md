# Tile Assets

Typically, Tiles are actually Sprites that are arranged on a Tilemap. In Unityâ€™s implementation, we use an intermediary Asset that references the Sprite instead. This enables us to extend the Tile itself in many ways, creating a robust and flexible system for Tiles and Tilemaps.
